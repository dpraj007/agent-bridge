## agent-bridge v0.1.0

Cross-agent session handoffs for **Claude Code** and **Grok Build**.

### Install

```bash
pip install "git+https://github.com/dpraj007/agent-bridge.git"
agent-bridge init
agent-bridge install-skills
agent-bridge sessions list
```

### Highlights

- Structured handoffs (`save` / `load` / `CURRENT.md`)
- Native session catalog (`sessions list|show|export`)
- Installable skills for both harnesses
- Local only, zero dependencies, MIT licensed
